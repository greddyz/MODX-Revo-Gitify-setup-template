MODX Revo Gitify setup template 
======

## Quick Installation

```` shell
$ git clone git@github.com:greddyz/modxSetup.git
$ composer install
$ chmod +x Gitify
$ gitify modx:install
$ gitify package:install --all
$ gitify build
````
